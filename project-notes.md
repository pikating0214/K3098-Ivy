# K3098 Project Notes

## Project

- Project name: Measure & Sound Adventure
- Target students: 6-year-old English learners
- Use: after-class review challenge
- Format: static HTML/CSS/JavaScript only
- Total score: 41 points

## Learning Goals

### Mathematics through English

- Order objects by height and length.
- Measure and compare length with non-standard units.
- Solve simple problems involving non-standard measurement.
- Practise:
  - `Which one is longer / shorter / taller?`
  - `Which one is the longest / shortest / tallest?`
  - `How long / tall is it?`
- Core words:
  - `long / longer / longest`
  - `short / shorter / shortest`
  - `tall / taller / tallest`

### Phonics

- Distinguish soft and hard sounds of `c` and `g`.
- Decode CVC words with the middle vowels `a, e, i, o, u`.
- Listen to a complete word and select the missing middle vowel.

## Game Structure

- Level 1: Length & Height Forest — 5 comparison and ordering tasks.
- Level 2: Block Measurement Lab — 5 non-standard measurement and problem-solving tasks.
  - The three-pipe comparison uses 4, 7, and 9 blocks and asks for the longest pipe.
- Level 3: Sound Detective Castle:
  - 16 soft/hard `c` and `g` sorting tasks in four separate groups.
  - 15 CVC listening tasks.
- The CVC words are mixed rather than grouped by vowel.
- Every CVC item uses a different order for the `a, e, i, o, u` answer buttons.
- Students must answer all 41 tasks before the final submit button unlocks.
- Final feedback shows the raw score, a five-star scale, and an original CSS cartoon reward.

## CVC Word Set

- `dog`, `pen`, `sun`, `map`, `sit`
- `fox`, `hen`, `cup`, `cat`, `fin`
- `log`, `bed`, `bus`, `jam`, `pig`

Each middle vowel appears three times.

## Soft and Hard C/G Word Set

- Soft C: `face`, `circle`, `city`, `rice`
- Hard C: `cut`, `cat`, `cup`, `corn`
- Soft G: `age`, `giant`, `gem`, `giraffe`
- Hard G: `dog`, `goat`, `game`, `gum`

## Feedback and Audio

- Correct answers are not shown before final submission.
- Before submission, incorrect selections receive a short knowledge-point hint.
- Every question has a browser speech button that reads its instruction or complete question.
- CVC audio uses free browser speech synthesis and reads the complete word.
- Object labels and measurement numbers sit below the illustrations rather than covering them.
- On phones, the four sound-category buttons stay visible above the word list in a compact 2-by-2 panel.
- No recorded audio, API, external service, or API key is used.
